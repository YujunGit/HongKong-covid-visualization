# 项目截图文件夹

这个文件夹用于存放项目的截图和演示图片。

## 建议的截图内容

1. **dashboard.png** - 整体大屏效果图
2. **map-feature.png** - 地图功能展示
3. **charts-overview.png** - 图表概览
4. **mobile-view.png** - 移动端响应式效果

## 如何添加截图

1. 启动项目：`python app.py`
2. 访问：http://localhost:8080
3. 截取屏幕截图
4. 保存到此文件夹
5. 在 README.md 中引用

## 图片格式建议

- **格式**：PNG 或 JPG
- **尺寸**：建议宽度 1200px 左右
- **文件大小**：控制在 1MB 以内
- **命名**：使用英文和连字符，如 `dashboard-overview.png`
